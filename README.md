# student_performance_web
# Öğrenci Performans Tahmin Sistemi (Student Performance Prediction)
Bu proje, makine öğrenmesi algoritmalarını kullanarak öğrencilerin akademik başarılarını tahmin eden uçtan uca (End-to-End) bir web uygulamasıdır.
Proje veri analizi, model eğitimi, modelin serileştirilmesi ve son olarak bir web arayüzüne (HTML/CSS) entegre edilerek canlıya alınması süreçlerini kapsar.
